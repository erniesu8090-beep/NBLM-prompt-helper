# CAST. Studio | NotebookLM 提示詞主控台

這是一個專為 **NotebookLM (NBLM)** 設計的提示詞配置與生成工具。本工具採用現代深色玻璃擬物風格（Dark Glassmorphic UI）設計，具備四個獨立分頁（雙通道語音、長簡報分組、影片總覽、簡報講解口白），能為您快速調配、生成、與複製最完美的 Prompt，極大化發揮 NotebookLM 的內容萃取與轉譯能力。

---

## 🚀 專案特點

* **零套件依賴 (Zero Dependency)**：僅使用原生 HTML5、CSS3 和 Vanilla JavaScript 撰寫。不需要安裝任何開發工具、編譯器或伺服器軟體，下載後**點擊即可在本機直接運行**，且支援完全離線操作。
* **精美擬物視覺 (Modern UI/UX)**：極具質感的深色毛玻璃背景，輔以動態呼吸霓虹（雙通道青/橘、事故警示紅、影片摘要紫）發光燈效，並內建點擊複製成功的綠色霓虹動畫反饋。
* **安全無隱私風險**：所有提示詞皆在您本機的瀏覽器中即時計算與渲染，**不會上傳任何個人設定或主題資料至第三方伺服器**。

## 🛠️ 四大功能模組介紹

### 🎙️ 1. 雙通道語音提示詞 (Audio Overview & Accident Analysis)
此功能為調配 NotebookLM 的雙人語音導讀（Audio Overview）而生，支援兩種運作模式：
* **一般主題導讀模式**：
  * **角色人設調製**：自訂兩位主持人的姓名、性別、立場倾向（如樂觀推廣 vs 保守懷疑）、語氣風格（如通俗科普 vs 犀利追問）、專業領域與口頭禪。
  * **節奏曲線設計**：支援 W 型起伏、漸強式、先高後穩、雲霄飛車等多種對話情緒張力走向。
* **🚨 事故案例解說模式 (安全分析融合)**：
  * 當在控制台開啟事故分析框架時，系統將自動注入專業事故調查的專屬 Prompt。
  * **安全分析框架**：支援 RCA 根源原因分析、Swiss Cheese 瑞士乳酪理論、5 Whys 連續發問分析、魚骨圖分析。
  * **解說目標**：可導向專業事故調查報告、員工安全培訓教材、或應急與 SOP 修訂建議，產出極具教育價值的語音解說腳本。

### 📊 2. 簡報 15頁限制突破 (Slide Helper)
為了解決 NotebookLM 在生成簡報時，單次最多只能生成約 13-15 頁的物理限制而設計。
* **分組規劃 Prompt**：提供專用的 NBLM 分組提示詞範本，引導 AI 將您的製程文件、檢修手冊自動評估最適總頁數，並切分為每組不超過 13 頁的簡報 Prompt。
* **首尾銜接機制**：自動命令 NBLM 在第 N 組的開頭，帶入前一組的視覺風格錨點與最後一頁的視覺緩衝，確保分組生成的簡報擁有 100% 一致的排版風格與邏輯。
* **PowerPoint 無損合併指引**：內建詳細的 PowerPoint **「重複使用投影片 (Reuse Slides)」** 操作步驟，教導如何勾選「保留來源格式」將分批下載的 PPT 檔案進行無損合併，保留全部動畫與母片樣式。

### 📹 3. 影片總覽提示詞 (Video Overviews & Nano Banana)
針對 NotebookLM 全新推出的「影片總覽 (Video Overviews)」功能所設計的客製化引導語配置工具。
* **觀看格式模式 (Format)**：
  * **Explainer (說明解說)**：著重深入、結構化且具邏輯性的觀念解說，由淺入深梳理前後因果脈絡。
  * **Brief (摘要速覽)**：精簡高效，以極高密度濃縮核心結論與重點金句，不拖泥帶水。
* **影像視覺風格 (Visual Style)**：搭配 Gemini 的 Nano Banana 影像生成模型，支援六種精緻的視覺插圖風格：
  * **Watercolor (水彩)**：柔和的水彩筆觸與渲染色調，呈現溫馨人文感視覺。
  * **Papercraft (紙藝)**：3D 立體摺紙與工藝雕刻視覺，具有獨特立體感。
  * **Anime (動漫風)**：日系二次元動漫插畫美學，色調明亮、線條精緻。
  * **Whiteboard (白板)**：簡潔手繪白板/黑板塗鴉線條，直觀且便於教育訓練。
  * **Retro Print (復古)**：懷舊報紙與網點印刷質感，呈現老電影般膠卷視覺。
  * **Heritage (古典)**：端莊優雅的歷史古典美學，羊皮紙般的古樸色調。
* **微調指令與受眾長度調配**：
  * **目標受眾與時長規劃**：自訂目標受眾群體（例如：國中生、新進員工）與預估影片長度（3-5 分鐘、5-8 分鐘、10-12 分鐘、15 分鐘以上）。
  * **客製微調指令 (Directives)**：輸入自訂敘事焦點與指示（例如：限定分析特定章節，或將食譜改編為易於跟做的流程影片），高度客製化影片敘事脈絡。

### 🗣️ 4. 簡報講解口白提示詞 (Slide Narrative Scripts)
專為投影片影片錄製與配音大綱設計，能為您產出逐頁的高品質讲解逐字稿。
* **角色人設調配 (Role)**：可自訂或選用「技術專家」、「科普講師」、「資深產品經理」、「理財顧問」、「說故事大師」等特定角色，並可自由配置其專業經歷背景。
* **講解調性與字數控制**：提供多種講解口吻（如專業嚴謹、生動風趣、故事引導、商業說服等），並可限制單頁字數（150字以內、50-300字、300-500字）以符合理想的簡報配音長度。
* **口白結構指南**：支援輸出「逐頁口白 + 重點整理」或「逐頁口白 + 轉場引導句」，抑或是完全無標記的「純口白逐字稿」以利 TTS 語音直接朗讀。

### 🌐 5. HTML 簡報格式化提示詞 (HTML Presentation Formatters)
專為網頁型互動簡報（包含 GSAP 物理動畫、CSS Keyframes 等動態網頁分鏡）所設計的規劃器。
* **視覺與排版規格化**：可根據來源資料自動或手動調整網頁風格（如 SaaS Dashboard 儀表板、未來賽博龐克網格、極簡商業簡練設計、粉筆白板手繪等）。
* **動畫與動態框架配置**：支援設定動態控制框架（如 GSAP 3 物理彈性、CSS 原生 Transitions、Lottie/Canvas 複雜向量等），並為每一頁 Scene 生成精準的進入、持續核心互動與退場 GSAP 程式碼動畫指示。
* **口白與字幕百分百對齊**：引導 NBLM 輸出時，將逐頁口白細切並標示精準的時間段與相應的字幕拷貝，非常適合搭配時光軸做自動字幕同步。

## 🏃 如何開始使用？

1. 在您的電腦中，**按兩下開啟 `prompt_helper.html`**。
2. 網頁開啟後，使用最上方的 Tab 按鈕在三個功能模組之間進行切換。
3. 根據您的需求調整參數，或直接使用右側的 **「快速模板 (Presets)」** 一鍵填入範例。
4. 點擊 **「📋 複製提示詞」** 按鈕，按鈕會閃爍綠色以示複製成功。
5. 前往 [NotebookLM 官方網站](https://notebooklm.google.com/)，將提示詞貼入對話框中即可使用！
