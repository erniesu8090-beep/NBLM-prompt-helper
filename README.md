# CAST. v2 | NotebookLM 提示詞主控台

這是一個專為 **NotebookLM (NBLM)** 設計的提示詞配置與生成工具。本工具採用現代深色玻璃擬物風格（Dark Glassmorphic UI）設計，具備三個獨立分頁（雙通道語音、長簡報分組、影片摘要），能為您快速調配、生成、與複製最完美的 Prompt，極大化發揮 NotebookLM 的內容萃取與轉譯能力。

---

## 🚀 專案特點

* **零套件依賴 (Zero Dependency)**：僅使用原生 HTML5、CSS3 和 Vanilla JavaScript 撰寫。不需要安裝任何開發工具、編譯器或伺服器軟體，下載後**點擊即可在本機直接運行**，且支援完全離線操作。
* **精美擬物視覺 (Modern UI/UX)**：極具質感的深色毛玻璃背景，輔以動態呼吸霓虹（雙通道青/橘、事故警示紅、影片摘要紫）發光燈效，並內建點擊複製成功的綠色霓虹動畫反饋。
* **安全無隱私風險**：所有提示詞皆在您本機的瀏覽器中即時計算與渲染，**不會上傳任何個人設定或主題資料至第三方伺服器**。

## 🛠️ 三大功能模組介紹

### 🎙️ 1. 雙通道語音提示詞 (Audio Overview & Accident Analysis)
此功能為調配 NotebookLM 的雙人語音導讀（Audio Overview）而生，支援兩種運作模式：
* **一般主題導讀模式**：
  * **角色人設調製**：自訂兩位主持人的姓名、性別、立場倾向（如樂觀推廣 vs 保守懷疑）、語氣風格（如通俗科普 vs 犀利追問）、專業領域與口頭禪。
  * **節奏曲線設計**：支援 W 型起伏、漸強式、先高後穩、雲霄飛車等多種對話情緒張力走向。
* **🚨 事故案例解說模式 (安全分析融合)**：
  * 當在控制台開啟事故分析框架時，系統將自動注入專業事故調查的專屬 Prompt。
  * **安全分析框架**：支援 RCA 根源原因分析、Swiss Cheese 瑞士乳酪理論、5 Whys 連續發問分析、魚骨圖分析。
  * **解說目標**：可導向專業事故調查報告、員工安全培訓教材、或應急與 SOP 修訂建議，產出極具教育價值的語音解說腳本。

### 📊 2. 簡報 15頁限制突破 (Slide Helper)
為了解決 NotebookLM 在生成簡報時，單次最多只能生成約 13-15 頁的物理限制而設計。
* **分組規劃 Prompt**：提供專用的 NBLM 分組提示詞範本，引導 AI 將您的製程文件、檢修手冊自動評估最適總頁數，並切分為每組不超過 13 頁的簡報 Prompt。
* **首尾銜接機制**：自動命令 NBLM 在第 N 組的開頭，帶入前一組的視覺風格錨點與最後一頁的視覺緩衝，確保分組生成的簡報擁有 100% 一致的排版風格與邏輯。
* **PowerPoint 無損合併指引**：內建詳細的 PowerPoint **「重複使用投影片 (Reuse Slides)」** 操作步驟，教導如何勾選「保留來源格式」將分批下載的 PPT 檔案進行無損合併，保留全部動畫與母片樣式。

### 📹 3. 影片摘要提示詞 (Video Summarizer)
針對影音檔案的逐字稿（Video Transcript）進行高效率的重點摘要與規格轉譯。
* **擷取焦點設定**：支援四種影片類型焦點優化：
  * **技術實操示範**：著重工具使用、操作手感、物理動作細節。
  * **化工/製程講解**：著重流體走向、閥門開度、自控邏輯與參數變化。
  * **安全與事故還原**：著重異常最初徵兆、人員避險防護與應急處置。
  * **學術/技術講座**：著重理論大綱、設備故障機制、頻譜分析數據與診斷結論。
* **輸出格式排版**：可自由選擇產出 **結構化時間軸筆記**、**操作重點與 SOP 清單**、或 **問答與培訓考核庫**。
* **口語錯字校正**：提示詞內建自動工程/化工名詞校正規則，防止逐字稿中的口語辨識錯誤干擾摘要品質。

## 🏃 如何開始使用？

1. 在您的電腦中，**按兩下開啟 `prompt_helper.html`**。
2. 網頁開啟後，使用最上方的 Tab 按鈕在三個功能模組之間進行切換。
3. 根據您的需求調整參數，或直接使用右側的 **「快速模板 (Presets)」** 一鍵填入範例。
4. 點擊 **「📋 複製提示詞」** 按鈕，按鈕會閃爍綠色以示複製成功。
5. 前往 [NotebookLM 官方網站](https://notebooklm.google/)，將提示詞貼入對話框中即可使用！
